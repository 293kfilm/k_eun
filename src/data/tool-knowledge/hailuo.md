================================================================
## Core Prompt Formula
================================================================

**[Camera Shot + Motion] + [Subject + Description] + [Action] + [Scene + Description] + [Lighting] + [Style/Mood]**

================================================================
## Component Breakdown
================================================================

### Camera Shot & Motion
- Shot type: close-up, wide shot, medium shot
- Dynamic movements: tracking, dolly zoom, drone shots

### Subject & Description
- Physical appearance
- Clothing
- Distinguishing features
- Emotional states

### Action
- Specific movements: "running", "jumping", "interacting with objects"

### Scene & Description
- Environment and setting context

### Lighting
- Type: bright, dark, natural, artificial, golden hour

### Style/Mood
- Tone: cinematic, dramatic, cheerful, film noir

## Character Consistency Best Practices
================================================================

- **Anchor Features** — distinctive accessories, hairstyles, clothing
- **Multi-Angle Details** — describe from various perspectives
- **Consistency Reinforcement** — repeat key descriptors across shots
- **Multiple Characters** — clearly distinguish each character's unique features

================================================================
## Advanced Cinematography Techniques
================================================================

### Dynamic Camera Movements
- Tracking/following shots (immersion)
- Establishing/reveal shots (scene introduction)
- POV / over-the-shoulder (viewer connection)

### Environmental Enhancement
- Weather + atmospheric elements
- Time-of-day (for lighting realism)
- Architectural/nature details

================================================================
## Prompt Categories
================================================================

### Narrative & Storytelling
- Character motivations
- Emotional arcs
- Action sequences
- Dialogue interactions

### Commercial & Marketing
- Product features
- Brand aesthetic alignment
- Clear calls-to-action

### Artistic & Creative
- Specific styles: "Pixar-style", "cyberpunk"
- Color grading preferences
- Special effects: slow-motion, time-lapse

================================================================
## Common Mistakes & Solutions
================================================================

| Issue | Solution |
|---|---|
| Vague descriptions | Specific, detailed language with context |
| Character inconsistencies | Anchor features + reinforce descriptors |
| Camera movement confusion | Simplify + clearer directional language |
| Style shifts | Consistent visual style throughout |
